# Toutiao Backend

FastAPI backend project for Toutiao-style application.

## Tech Stack

- FastAPI
- SQLAlchemy
- MySQL
- Redis
- Python

## Environment Variables

Create a `.env` file based on `.env.example`.

```env
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=toutiao

REDIS_HOST=localhost
REDIS_PORT=6379

SECRET_KEY=your_secret_key
